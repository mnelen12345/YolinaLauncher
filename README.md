# YolinaLauncher
YolinaLauncher - is a game launcher program from the company hoyoverse

## Installation
```
pip install pypresence Flask psutil pywebview

git clone https://github.com/mnelen12345/YolinaLauncher.git
cd YolinaLauncher
python base.py
```
